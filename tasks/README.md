# Tasks

This folder contains phase-based implementation tasks for CueSpark Interview Coach.

Each task is intended to be given to Codex or another coding agent one at a time.

## Rule

Do not ask an agent to implement the whole project at once. Use one task file at a time.

Recommended prompt:

```text
Read AGENTS.md, docs/00-project-overview.md, docs/01-architecture.md, docs/02-backend-design.md, and this task file. Implement only this task. Do not add out-of-scope features. Report changed files and verification steps.
```

## Phases

```text
phase0-foundation
phase1-session-documents
phase2-interview-engine
phase3-answer-evaluation
phase4-report-frontend
phase5-future-scope
```
